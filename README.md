# Guess My Number

Number-guessing game built with HTML, CSS, and JS.

## Rules

- Guess a number between 1 and 20.
- You have 20 guesses each round.
- Every guess will cost you 1 point.
- The round will end when you run out of guesses or when you guess the right number.
- Your starting high score will be 0. It will be updated with your highest score for any round.
